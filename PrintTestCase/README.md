# Print Test Case
1つの整数を標準入力から読み込みます。
0を入力すると読み込みを終了し、入力されたデータを出力します。

# Requirement
* C#

# Usage
```bash
dotnet run print-test-case
// 1
// 2
// 3
// 4
// 5
// 6
// 7
// 9
// 0
// Case 1: 1
// Case 2: 2
// Case 3: 3
// Case 4: 4
// Case 5: 5
// Case 6: 6
// Case 7: 7
// Case 8: 9
```
